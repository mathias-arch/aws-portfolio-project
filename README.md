# aws-portfolio-project
Infraestructura Cloud para gestión de entornos corporativos. Utiliza Amazon EC2 para el procesamiento de peticiones, S3 como almacenamiento persistente y RDS para la integridad de los registros. Diseñado bajo seguridad IAM y optimización de recursos, con despliegue desde GitHub y desarrollo asistido por IA en Antigravity.
